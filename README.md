# 密码生成器

一款简洁美观的密码生成工具，支持 Windows 7/10/11。

## 功能特性

- ✅ 自定义字符类型：大写字母、小写字母、数字、特殊字符
- ✅ 自定义密码长度：8-32 位
- ✅ 一键复制到剪贴板
- ✅ 防重复生成
- ✅ 无需管理员权限
- ✅ 不收集任何用户数据

## 运行方式

### 方式一：直接运行 Python 脚本
```bash
python password_generator.py
```

### 方式二：打包成 exe（推荐）
```bash
python build_exe.py
```
打包完成后，exe 文件位于 `dist/密码生成器.exe`

## 系统要求

- Windows 7/10/11（32位/64位）
- Python 3.6+（仅开发时需要）

## 默认配置

- 默认勾选：大写字母 + 小写字母 + 数字
- 默认长度：12 位
